# Retail Sales Analysis & Prediction

## Objective
Analyze retail sales data and build a simple prediction model.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Analysis Performed
- Data Cleaning
- Sales Trend Analysis
- Regional Analysis
- Profit Analysis
- Correlation Analysis
- Sales Prediction

## Conclusion
The project identified sales trends, profitable regions, and factors influencing revenue using real-world retail data.
